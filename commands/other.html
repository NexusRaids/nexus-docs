Other¶

help¶

Command

Description

!help <command>

Use to get info on a command or category. If no <command> is given a list of all commands is returned.

pst¶

Command

Description

![pt|pdt|pst]

Get the current time in Pacific Time. This is useful for events that start in this timezone.

summon¶

Command

Description

![summon|ping]

Summon Nexus. Or get the latency of the bot. Whatever sounds cooler.

support¶

Command

Description

!support

This will give you the invite link to the support discord server of Nexus.
