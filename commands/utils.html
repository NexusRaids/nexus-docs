pvp-rank¶

Command

Description

![pvp|pvp-rank|pvp-rankings] <pokemon> <league=great-league> <category=overall>
Use this to show the PvP rankings from PvPoke.com.
The following leagues are available: great-league|great|gl, ultra-league|ultra|ul and master-league|master|ml.
The following categories are available: overall|ov, leads|le, closers|cl, switches|sw, chargers|ch, attackers|at and consistency|co.
