Profile¶

Profiles in Nexus are shared across servers wherever the bot is installed allowing for a seamless experience in every community.

change-profile¶

Command

Description

!change-profile

Use this to change your profile. You need to support Nexus to access this command.

ign¶

Command

Description

!ign <user>

Use this to retrieve someone’s in-game name.

level¶

Command

Description

![level|lvl] <user>

Use this to retrieve someone’s level.

profile¶

Command

Description

!profile <user>

Use this to show someone’s profile.

save-profile¶

Command

Description

!save-profile

Use this to save your current profile. You need to support Nexus to access this command.

search¶

Command

Description

!search <ign>

Use this to search for a trainer in your server.

set¶

Command

Description

!set country <country>

Use this to set your country. You can give this the name of the country or the country code. (e.g. United States or US).

!set ign <ign>

Use this to set your in-game name.

!set [level|lvl] <level>

Use this to set your level. You can set it to a maximum of 50.

!set location <latitude> <longitude>

Use this to set your location for trading purposes. Your location can’t be seen by anyone. You need to input your latitude and longitude.

!set profile

Use this to set all settings in your profile.

!set [tc|fc|trainer-code|friend-code] <trainer_code>

Use this to set your trainer code.

!set team <team>

Use this to set your team. The available teams are mystic, valor and instinct.

trainer-code¶

Command

Description

![tc|fc|trainer-code|friend-code] <user>

Use this to retrieve someone’s trainer code.

update-my-roles¶

Command

Description

!update-my-roles

Use this to update your profile roles according to your current information set on Nexus. If your server has no profile roles, this command will not do anything.
